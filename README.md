react-via-cep
=======

Robô para carragar suas notas do sigaa

## Configuração 

  Cire uma arquivo com o nome .env e o conteúdo de .env.example. Preencha os com seu usuário e senha.

### Como usar (NPM)
Instale os pacotes:
```
npm i
```
Inicie o bot

```
npm start
```

### Como usar (YARN)
Instale os pacotes:
```
yarn
```
Inicie o bot

```
yarn start
```
